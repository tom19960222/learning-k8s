#### Step 2.2: 4 MDX

每個 MDX:
- `layout: doc`、`title: Ceph — <主題>`
- 場景優先；ASCII 圖；code block 第一行 `// File: <path>` 或 `/* File: <path> */`
- URL 用 `https://github.com/ceph/ceph/blob/v19.2.3/...`
- forbidden: import statements、`<QuizQuestion>`、大陸用語、杜撰 symbol、`<X>` 之類的 angle bracket placeholder（會被 MDX 當 JSX）

骨架見 spec § 4 頁切法。

**注意 MDX `<` 字元**：所有不在 backtick / code block 內的 `<...>` 都會被當 JSX；如果要寫 placeholder 用 `[X]` 或 `&lt;X&gt;` 而非 `<X>`。

```bash

echo $! > /tmp/sp6-dev.pid; sleep 8
PORT=$(grep -oE 'localhost:[0-9]+' /tmp/sp6-dev.log | tail -1 | cut -d: -f2)
curl -fsS http://localhost:$PORT/ceph/ | grep -c -i "ceph\|RADOS"
curl -fsS http://localhost:$PORT/ceph/features/architecture/ | grep -c -i "MON\|OSD\|RADOS"

kill $(cat /tmp/sp6-dev.pid) 2>/dev/null; sleep 1
```
